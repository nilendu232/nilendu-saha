Overview

Designed a custom Eagle-based, application-specific controller PCB with an onboard microcontroller, sensor inputs, and outputs for relays and DC motors to run an automated unit sorting system.

Highlights

  • Designed schematics which demonstrated the layout of sensor inputs plus relay and motor output terminals.
  • Developed a 2-layer board based on the schematics .
  • Reduced cabinet wiring by ~15–20% by consolidating secured I/O connectors.
  • Improved assembly time by ~25% via repeatable harnessing and clearer integration points.
  • Developed control software enabling autonomous operation and stable throughput (~1 unit per 2 seconds) into the main line.

Role & responsibilities

  • Captured requirements (I/O count, supply rails, noise/EMI constraints, connector strategy).
  • Designed schematics and selected parts (MCU, protection, drivers, connectors, regulation).
  • Created a 2-Layer PCB layout for robust power + signal routing and serviceability (test points, labels).
  • Performed bring-up/testing and supported machine commissioning and debugging.
  • Harnessed the features of ESP32 and made it the heart of the PCB.
  • Developed firmware/software to control the sorting cycle and run the system autonomously.

End result

  • 15–20% less wiring and 25% faster assembly compared to the previous build.
  • Achieved stable throughput contributing 1 unit per 2 seconds to the production line.
  • Implemented Horizontal Scaling for better throughput and reliability.
