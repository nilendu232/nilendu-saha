Overview

Engineered the machine’s complete electronics integration and embedded control software on the Duet3D platform to deliver high-throughput, dimension-based product sorting. Implemented clear machine-state indication using a safety status LEDs (boot/ready/running/fault) for fast on-floor visibility. Achieved ~98% sorting accuracy while sustaining 40,000–50,000 units/hour, and improved separation quality to help recover/recycle ~80% of key resources.

Highlights

  • Delivered dimension-based sorting with ~98% accuracy at 40,000–50,000 units/hour.
  • Integrated Duet3D control platform to coordinate sensors, actuators, and sequencing logic.
  • Added a safety status LEDs to convey machine state at a glance.
  • Implemented robust fault handling (jam detection, sensor checks) with guided recovery.
  • Designed for easy maintenance using modular wiring and clear I/O mapping.
  • Improved health and safety via controlled-stop behavior, restart rules, and safer maintenance procedures.
  • Trained operators and technicians for consistent operationand faster troubleshooting.

Role & responsibilities

  • Captured requirements (throughput, accuracy, I/O, environment constraints, safety expectations).
  • Engineered machine electronics integration: power distribution, protection, and structured I/O allocation.
  • Developed embedded control logic on Duet3D along with the easy access web interface.
  • Implemented machine-state indication using safety LEDs and consistent fault messaging.
  • Commissioned on-machine: calibration, tuning, reliability testing, and production handover support.
  • Created documentation (I/O maps, operating procedure, troubleshooting guide, maintenance checklist).
  • Provided training to operators and technicians for safe operation and maintenance.

End result

  • Achieved stable throughput of 40,000–50,000 units/hour with ~98% sorting accuracy.
  • Reduced downtime through diagnostics, fault handling, and maintenance-friendly design choices.
  • Improved safety and clarity with a safety LEDs indicating machine status.
  • Enabled ~80% recovery/recycling of key resources through improved separation quality.
